# uvlab

[English](./README.md) | 日本語

`uv` のパワーを最大限に活用した、ミニマリストのための JupyterLab テンプレート。
環境構築に時間を溶かすのをやめ、コードを書くことに集中するための道具箱です。

## 特徴

- **環境の完全隔離**: 各ラボ（`lab_*.py`）ごとに独立した仮想環境を `uv` が瞬時に構築。
- **設定のポータビリティ**: `notebooks/` をルートとする設定がリポジトリ内で完結。
- **インテリジェント・ランチャー**: ファイル内の DocString を解析し、メニュー形式でラボを起動。
- **クリーンな運用**: ブラウザ利用に特化し、`Ctrl+C` でポートを確実に解放。

## 使い方

### 1. 準備
[uv](https://github.com/astral-sh/uv) がインストールされていることを確認してください。

### 2. 起動
リポジトリのルートでランチャーを実行します。

```bash
uv run.py
```

### 3. 新しいラボ環境の追加

`lab/lab_sample.py` をコピーして新しいファイル（例: `lab_ai.py`）を作成し、冒頭の `dependencies` を書き換えるだけです。

## ディレクトリ構成

* `uvlab.py`: ラボを起動するための CLI ランチャー
* `lab/`: ラボの定義ファイルと Jupyter 設定を集約
* `notebooks/`: 全ての `.ipynb` ファイルを保存する作業スペース

## 注意事項

* **ブラウザ推奨**: VSCode の Jupyter Server 機能は認証やプロセスの管理で不安定になる可能性があるため、ブラウザ（`http://localhost:{port}/lab`）での利用を強く推奨します。
